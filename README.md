# Hicheel 2: Tournament Registration System with Optimistic UX

This project demonstrates:

- Server-first state management with Next.js server actions
- Form -> server -> SQLite flow
- Optimistic UX with rollback-safe feedback
- SSR admin dashboard with cached leaderboard data
- Business reasoning about trust, conversion, and perceived speed

## Routes

- `/` registration page for parents and players
- `/dashboard` admin dashboard with live registrations

## Run

```bash
npm install
npm run dev
```
